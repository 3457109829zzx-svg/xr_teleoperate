本项目来源于宇树开源项目[xr_teleoperate](https://github.com/unitreerobotics/xr_teleoperate)，在其基础上根据自己的需求进行了优化适配，以下是新增功能：

## 新增功能

### 1. PICO 手柄全流程控制（5.18 / 5.24）

告别键盘，所有操作通过 PICO 手柄完成：

| 按键 | 功能 |
|------|------|
| **Y**（左手 B） | 启动遥操作 / 暂停 / 恢复 |
| **X**（左手 A） | 退出程序 |
| **B**（右手 B） | 开始 / 结束录制 |

- 暂停后不退出程序，可再次按 Y 恢复，无需重启
- 恢复时自动重新标定 IK 偏移，确保末端位置不跳变

### 2. 手柄扳机直接控制灵巧手（5.19）

- 通过 PICO 手柄扳机直接控制 Inspire 灵巧手开合，绕过手势重定向
- 扳机松开时自动切回手势重定向，两者无缝切换
- 兼容旧版，不传参数时不影响原有逻辑

### 3. 启动/恢复时末端位置防突变（5.23）

- 新增 IK 偏移标定机制，每次进入遥操作前自动记录偏移量
- 彻底解决机械臂启动瞬间末端跳变的问题

### 4. 录制体验优化（5.25）

- **最大录制数限制**：`--max-episodes` 参数，防止误操作导致数据膨胀
- **一键删除失败 episode**：录制不理想可直接删掉当前条，不留脏数据
- **头显状态覆盖层**：XR 头显内实时显示录制状态（红灯 + episode 编号）和采集进度
